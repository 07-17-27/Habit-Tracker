# Habit Tracker 📚

A clean, modern, production-quality habit tracking web app built with **vanilla HTML, CSS, and JavaScript** — no frameworks, no dependencies.

## ✨ Features

- **Monthly Calendar** — Full 7-column grid, auto-generated per month/year
- **Multi-Habit Tracking** — Add unlimited habits with custom names and emojis
- **Toggle Days** — Click any past day to mark it complete (animated)
- **Statistics** — Total completed days, current streak, best streak, monthly %
- **Progress Bar** — Animated gradient bar showing monthly completion
- **Month Navigation** — Browse any past or future month
- **Dark Mode** — Smooth light/dark toggle, preference saved
- **Data Persistence** — localStorage keeps all data across page refreshes
- **Reset Button** — Clear a habit's data with confirmation
- **Fully Responsive** — Works on mobile and desktop

## 🚀 Getting Started

### Run locally

No build step needed — just open the file:

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/habit-tracker.git
cd habit-tracker

# Open in browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Or use VS Code's **Live Server** extension for hot reload.

### Deploy to GitHub Pages (free hosting)

1. Push to GitHub
2. Go to **Settings → Pages → Source: main branch → Save**
3. Your app is live at `https://YOUR_USERNAME.github.io/habit-tracker`
